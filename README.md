## Usage

### Node

Use the following script to quickly create a **Node** or **Node Library** project.


```bash
/bin/bash -c "$(curl -fsSL https://now.box.toys/bin/node)"
```

```bash
/bin/bash -c "$(curl -fsSL https://now.box.toys/bin/node.part)"
```
